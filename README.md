# U8 直报直调提词器 V2 (Teleprompter V2)

![预览图](http://youke.xn--y7xa690gmna.cn/s1/2026/01/20/696ee410792e2.webp)

U8 直报直调提词器是一款专为演讲、录课、直播设计的专业网页版提词工具。版本 V2 带来了全新的全屏交互体验和更精美的视觉设计。

## 🌟 核心特性

- **专业文本处理**：支持 `.txt` 文件导入，具备智能分段、格式化文本功能。
- **全屏沉浸模式**：一键切换全屏，专为专业提词场景优化，消除视觉干扰。
- **悬浮交互控制**：全屏模式下自动感应悬浮控制栏，包含：
  - **播放/暂停**：实时控制滚动状态。
  - **重置**：一键返回开头。
  - **智能隐藏**：鼠标静止 3 秒后自动淡出，移动即刻唤醒。
- **极致显示控制**：
  - 自由调节滚动速度（0.1 - 200）。
  - 实时调整字体大小、行间距。
  - 支持**横向镜像**显示，适配提词器硬件反射镜片。
  - 自定义文字颜色与背景颜色。
- **现代化 UI 设计**：基于 Glassmorphism（玻璃拟态）风格，提供极致的使用体验。

## ⌨️ 快捷键支持

- `Space` (空格)：开始 / 暂停滚动
- `R`：重置滚动位置
- `F`：切换全屏模式
- `Esc`：退出全屏

## 🛠️ 技术栈

- **HTML5**: 语义化结构
- **CSS3**: 原生 CSS 变量、Flexbox 布局、玻璃拟态特效、持久化动画
- **Vanilla JavaScript**: 高性能滚动逻辑，无第三方库依赖

## 🚀 快速开始

1. 克隆本项目到本地。
2. 双击 `index.html` 即可在浏览器中打开使用。
3. 导入你的 `.txt` 演讲稿。
4. 点击“格式化文本”并按下“开始”或空格键。
